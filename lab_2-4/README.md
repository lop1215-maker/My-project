## Лабораторная работа 2-3: Одномерные динамические массивы

### 1. Задание:
Создать программу для анализа сенсора температуры, которая:
1. Считывает N измерений температуры в динамический массив
2. Анализирует данные (среднее, минимум, максимум)
3. Создаёт новый массив со значениями выше среднего
4. Использует только указатели и арифметику указателей

### 2. Описание алгоритма:
- **'inputData()'** - заполняет массив значениями с клавиатуры
- **'average()'** - вычисляет среднее значение массива
- **'minValue()'** - находит минимальное значение
- **'maxValue()'** - находит максимальное значение
- **'filterAboveAverage()'** - фильтрует значения выше среднего
- Все операции выполняются через арифметику указателей

### 3. Ключевые моменты:
- Использование 'new' и 'delete[]' для управления памятью
- Передача массивов в функции через указатели
- Работа с элементами через '*(p + i)' вместо 'p[i]'
- Правильное освобождение памяти

## Лабораторная работа 4: Многомерные динамические массивы

### 1. Задание:
Создать программу для анализа температурных данных по нескольким дням:
1. Работа с двумерным массивом (дни и измерения в день)
2. Вычисление средней температуры за день
3. Вычисление общей средней температуры
4. Вывод дней с температурой выше средней

### 2. Описание алгоритма:
- **'inputData()'** - заполняет двумерный массив значениями
- **'dayAverage()'** - вычисляет среднюю температуру за один день
- **'overallAverage()'** - вычисляет общую среднюю температуру
- **'showHotDays()'** - выводит дни с температурой выше средней
- Использование указателей на указатели для двумерных массивов

### 3. Ключевые моменты:
- Создание двумерного массива: 'float** data = new float*[d]'
- Каждая строка создаётся отдельно: '*(data + i) = new float[h]'
- Доступ к элементам через двойное разыменнование: '*(*(p + i) + j)'
- Правильное освобождение памяти в обратном порядке

  ---

  ## Технические особенности:

  ### Для лабораторных 2-3:
- float* data = new float[n];       // Создание динамического массива
- *(p + i) = value;                  // Запись
- float avg = average(data, n);      // Использование
- delete[] data;                     // Удаление

  ### Для лабораторной 4:
- float** data = new float*[d]       // Создания массива указателей
- for (int i = 0; i < d; i++){
  *(data + i) = new float[h];        // Заполнения двумерного массива указателями на другие массивы
  }
- cin >> *(*(p + i) + j);            // Чтение элемента
