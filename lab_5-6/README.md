# Library project

# Система управления личной библиотекой

Консольное приложение для управления библиотекой на C++ с ООП.

## Сборка и запуск
mkdir build
cd build
cmake ..
cmake --build .
./library


## Функционал
- [x] Просмотр книг и пользователей
- [x] Добавление книг/пользователей  
- [x] Выдача/возврат книг
- [x] Поиск по ISBN
- [x] Сохранение в файл
