1. Задание
Реализовать шаблонную функцию groupStats(arr, n, keyFunc, valueFunc), которая по массиву arr размера n формирует группы элементов по значению ключа keyFunc(x) и для каждой группы считает статистику: ключ группы, количество элементов, минимальный и максимальный элемент, сумму и среднее значение по valueFunc(x). Показать работу функции на массивах int, double и std::string с разными правилами группировки.
​

2. Описание алгоритма
Функция groupStats принимает массив, его размер и две функции: keyFunc для получения ключа и valueFunc для значения, которое идёт в сумму. Для каждого элемента массива вычисляется ключ и значение, затем выполняется поиск существующей группы с таким же ключом во векторе групп; если группа найдена — обновляются счётчик, минимум, максимум, сумма и среднее, если нет — создаётся новая запись GroupInfo с начальными значениями. Шаблонная структура GroupInfo<T> хранит ключ, количество элементов, минимальный и максимальный элементы типа T, а также сумму и среднее (double).
​

В main заданы три примера: для массива int ключом является последняя цифра числа, для double — номер интервала (меньше 0, от 0 до 10, больше 10), для std::string — первая буква строки, а значением служит длина строки. Для каждого примера вызывается groupStats, и затем по всем группам выводятся поля: key, count, minElem, maxElem, sum, avg.
​

3. Код
Код программы находится в файле main.cpp и реализован на языке C++ с использованием шаблонной функции groupStats и шаблонной структуры GroupInfo<T>, а также стандартных контейнеров и строк (<vector>, <string>, <iostream>). В качестве функций keyFunc и valueFunc используются обычные функции: для int — получение последней цифры и самого числа, для double — определение интервала и само число, для string — первая буква и длина строки.
